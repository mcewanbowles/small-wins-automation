# Black-and-White Outline Images Folder

This folder contains black-and-white outline images for coloring activities.

## Guidelines

- **File Format**: PNG recommended
- **Style**: Clear black outlines on white/transparent background
- **Resolution**: At least 1000x1000 pixels
- **Line Weight**: Bold, clear lines suitable for coloring
- **Naming**: Use descriptive names (e.g., `apple_outline.png`, `dog_outline.png`)

## Usage

These images are loaded using:
```python
folder_type='bw_outline'
```

## Used By

- Coloring Strips Generator
- Coloring Sheets Generator

## Examples

Place your outline images here:
- `apple_outline.png` - Simple apple outline
- `dog_outline.png` - Dog outline for coloring
- `flower_outline.png` - Flower outline

These should be simple, clear outlines that students can easily color within the lines.
