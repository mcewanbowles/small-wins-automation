# LOGO USAGE STRATEGY

## Small Wins Studio - Dual Logo System

---

## LOGO FILES

### Star Logo ⭐ (Frequent Use)
**File:** `assets/branding/small_wins_logo.png/star.png`

**Description:** Star symbol only, compact and clean

**Use:** Frequent use when text accompanies the logo

### Full SMS Logo (Occasional Use)
**File:** `assets/branding/small_wins_logo.png/sms.png`

**Description:** Complete logo with "Small Wins Studio" text

**Use:** Occasional use for standalone placements

---

## WHEN TO USE EACH LOGO

### ⭐ Star Logo (FREQUENT USE)

**Use When:**
- Text accompanies the logo
- Space is limited
- Clean, modern look desired
- Business name is already written

**Perfect For:**
- **Product Covers** - Next to "Small Wins Studio" text
- **Footers** - Small, next to copyright notice
- **Headers** - When brand name is present
- **Page Titles** - Alongside text
- **Compact Spaces** - Professional appearance

**Placement Examples:**
```
[⭐] Small Wins Studio
BROWN BEAR MATCHING - LEVEL 1
```

```
© 2025 [⭐] Small Wins Studio • PCS® symbols...
```

---

### 📱 Full SMS Logo (OCCASIONAL USE)

**Use When:**
- Logo stands alone without text
- Large hero placements
- Brand identity focus
- Marketing materials

**Perfect For:**
- **Social Media Profiles** - Profile images
- **Business Cards** - Standalone branding
- **Email Signatures** - Professional identity
- **Marketing Materials** - Promotional items
- **Hero Sections** - Large feature areas
- **Watermarks** - Full brand protection

**Placement Examples:**
```
┌─────────────────┐
│                 │
│  [FULL LOGO]    │
│  Small Wins     │
│  Studio         │
│                 │
└─────────────────┘
```

## DESIGN SPECIFICATIONS

### Star Logo Sizing

**Product Covers:**
- Height: 60-80px
- Next to brand text
- Top left or center
- Professional spacing

**Footers:**
- Height: 20-30px
- Before copyright text
- Small and subtle
- Maintains brand

**Headers:**
- Height: 40-50px
- With page title
- Clean appearance

**General:**
- Transparent background
- PNG format
- High resolution (300 DPI)
- Scales proportionally

---

### Full Logo Sizing

**Standalone Use:**
- Width: 200-400px
- Centered placement
- Generous spacing
- Brand focus

**Marketing Materials:**
- High resolution required
- 300+ DPI for print
- Transparent background
- Professional quality

## IMPLEMENTATION

### Generators Updated

**All generators now use smart logo selection:**

```python
# For covers and pages (with text)
star_logo = "assets/branding/small_wins_logo.png/star.png"

# For standalone use
full_logo = "assets/branding/small_wins_logo.png/sms.png"
```

**Automatic Logic:**
- Covers: Star logo (text present)
- Footers: Star logo (compact)
- Headers: Star logo (clean)
- Marketing: Full logo (standalone)

---

## BEST PRACTICES

### DO:
✅ Use star logo with text
✅ Use full logo standalone
✅ Maintain aspect ratios
✅ Use transparent backgrounds
✅ Keep high resolution
✅ Professional placement
✅ Consistent sizing
✅ Test on different backgrounds

### DON'T:
❌ Stretch or distort logos
❌ Use low resolution
❌ Change colors (unless BW version)
❌ Add effects or shadows
❌ Use star without nearby text
❌ Use full logo with redundant text
❌ Place on clashing backgrounds
❌ Make too small to see
