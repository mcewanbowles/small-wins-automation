# Deprecated Generators

This folder contains **quarantined** generators that are no longer actively maintained but have been preserved for reference.

## Purpose
Generators are moved here (not deleted) when:
- They are superseded by newer implementations
- They are temporarily disabled pending review
- They contain experimental code not ready for production

## Contents
No generators are currently quarantined. When generators are deprecated:
1. Move the entire folder here
2. Add an entry to this README
3. Do NOT delete the original code

## Restoration
To restore a deprecated generator:
1. Move the folder back to `/generators/`
2. Update `/docs/product-type-standards.md`
3. Ensure it follows current standards before use
