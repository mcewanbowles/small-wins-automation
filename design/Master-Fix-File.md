# Master Fix File — Small Wins Studio

This file defines the universal corrections and refinements that must be applied to all generated outputs.

---

## 1. Title Block Fixes
- Ensure accent stripe height is correct.
- Ensure title and subtitle sit fully inside the stripe.
- Remove level from title.
- Center text vertically and horizontally.

---

## 2. Icon Fixes
- Increase icon size if page has unused space.
- Reduce padding inside boxes.
- Ensure icons are crisp and centered.
- Confirm all icons from the theme are included.

---

## 3. Matching Layout Fixes
- Increase spacing between columns.
- Increase spacing between rows.
- Ensure boxes are balanced and centered.

---

## 4. Watermark Fixes
- Level 1: add watermark at correct opacity and scale.
- Levels 2–4: remove watermark.

---

## 5. Target Box Fixes
- Remove drop shadow.
- Add inner stroke (2–3 px).
- Optional: soft inner glow (5–8% opacity).
- Increase target image size.

---

## 6. Cutout Fixes
- Ensure 5 icons per strip.
- Icons touch edges.
- Rounded corners.
- Include all icons.

---

## 7. Storage Label Fixes
- Ensure 3-column vocabulary table.
- Include all icons.
- Align text and icons cleanly.

---

## 8. Output Fixes
- Regenerate color and BW PDFs.
- Save to correct folder.
- Overwrite existing files.
