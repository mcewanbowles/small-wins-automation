# Small Wins Studio - Design Constitution

This document defines the universal visual, structural, and accessibility standards for all Small Wins Studio resources. All generators must comply with these rules unless explicitly overridden for a specific product type.

---

## 1. Products

| Product | Pack Code | Generator | Quick Start |
|---------|-----------|-----------|-------------|
| **Matching** | BB03 | `generate_matching_constitution.py` | `generate_quick_start_professional.py` |
| **Find & Cover** | BB04 | `generate_find_cover_constitution.py` | `generate_quick_start_professional.py` |
| **Bingo** | BB02 | `generate_bingo_constitution.py` | `generate_quick_start_professional.py` |
| **AAC Literacy Chat Boards** | BB01/BB05 | `generate_aac_constitution.py` | N/A |

All products have 5 differentiated levels and ship with Color + B&W versions.

---

## 2. Page Structure

- **Page size:** US Letter (8.5" x 11") portrait; landscape for AAC boards
- **Border:** Light blue `#A0C4E8`, 3pt stroke, 0.18" corner radius, 0.25" margin from page edge
- **Font:** Comic Sans MS throughout (Bold for titles, Regular for body)

---

## 3. Footer (Gold Standard)

Two centered lines in grey `#999999`:

- **Line 1 (bold, 9pt):** `{Product Title} | Level {X} | {Pack Code} | Page {Y}/{Z}`
- **Line 2 (regular, 8pt):** `Small Wins Studio | PCS symbols used with active PCS Maker Personal License. | © 2026`

> Do NOT duplicate the business name on line 1. Line 2 handles branding.

---

## 4. Title Block / Accent Stripe

- Accent stripe sits inside the border with rounded corners (0.12" radius)
- **Accent stripe color:** Level-specific (see Section 5)
- Title: centered, white, Comic Sans Bold 22pt
- Subtitle: centered, white, Comic Sans 14pt
- Stripe height: ~1.0" (300 DPI) for activity pages

---

## 5. Level Color Coding (UNIVERSAL)

These colors are CONSISTENT across ALL products (covers, labels, page headers, accent stripes):

| Level | Color | Hex Code | Name |
|-------|-------|----------|------|
| **L1** | Orange | `#F4A259` | Errorless |
| **L2** | Blue | `#4A90E2` | Supported / Distractors |
| **L3** | Green | `#7BC47F` | Independent / Moderate |
| **L4** | Purple | `#9B59B6` | Generalisation / Challenge |
| **L5** | Red | `#E74C3C` | Extension |

> The accent stripe color instantly tells teachers which level they're using.

## 6. Quick Start Guide Standards

One Quick Start per product (not per level). Generated by `generate_quick_start_professional.py`.

### Layout
- **Single page**, US Letter portrait
- **Header:** Small Wins logo (top-left, 1.6" wide) + teal `#20B2AA` accent bar (top-right)
- **Body:** 2-column layout with 6 sections per column
- **Footer:** Gold standard (grey, 2 lines)
- **Border:** Light blue `#A0C4E8` rounded rectangle

### Typography
- Section headings: Comic Sans Bold 12pt, teal, with underline
- Body text: Comic Sans 10pt, dark grey `#333333`
- Line height: 15pt (boxes), 15pt (bullets), 16pt (numbered steps)
- Section gap: 12pt between sections
- Heading gap: 14pt after underline before content

### Content Boxes
- **Grey box** (`#F8F9FA`): teal left border, for descriptions and prep info
- **Yellow highlight** (`#FFFBEA`, gold `#FFD700` border): for series callouts and next steps
- **Blue info box** (`#E6F7FF`, blue `#1890FF` left border): for AAC/communication tips
- **Numbered steps:** teal circle badges with white numbers
- **Bullet lists:** teal bullets for bold-prefix items, dark grey for plain items

### Sections (per product)
Left column: What this resource is, Part of a Differentiated Series, What's included, Set-up steps, Prep, Student routine
Right column: Teaching support (prompting ladder), Communication + AAC, Ways to use, Troubleshooting, Next steps, Quick games / Prep & Storage

---

## 7. Icon Standards
- All icons sourced from `/assets/{theme}/icons/`
- Icon style: consistent stroke, color, and proportions
- Icon scaling:
  - Target images: 1.8" box, centered
  - Matching boxes: 1.0" x 1.0" box, 0.75 scale inside
- Icons must be crisp, centered, and never distorted.

---

## 8. Matching Box Standards
- Box size: 1.0" x 1.0"
- Rounded corners: 0.1"
- Velcro dot: 0.35" diameter, centered, light grey (#E6E6E6)
- Vertical spacing: 0.15" between rows
- 5 rows per page (5x2 layout)

---

## 9. Watermark Logic
- Level 1 only:
  - Watermark of the correct icon
  - Opacity: 25%
  - Scale: 70% of box
  - Centered behind velcro dot
- Levels 2-5: no watermark

---

## 10. Cutout Standards
- 5 icons per strip
- Icons touch the edges of their boxes
- Rounded corners
- Title: "Cutout Matching Pieces"
- Accent stripe: level-specific color
- Include all icons used in the activity

---

## 11. Storage Labels
- Title: "Storage Labels - {Product} Pack"
- Pack code displayed
- 3-column vocabulary table
- All icons included
- Clean alignment and spacing
- Accent stripe: level-specific color

---

## 12. Color System
- **Brand teal:** `#20B2AA` (Quick Start headings, accents)
- **Border blue:** `#A0C4E8` (page borders)
- **Navy:** `#1E3A5F` (activity page titles)
- **Level colors:** see Section 5
- **Text:** dark grey `#333333` (body), grey `#999999` (footer)
- **Backgrounds:** white `#FFFFFF`, section grey `#F8F9FA`

---

## 13. TpT Packaging Standards

Each level ships as a ZIP containing:
- Color PDF (activity pages)
- B&W PDF (activity pages)
- Terms of Use PDF
- Quick Start PDF (one per product, same in every level ZIP)
- Storage Labels PDF (Bingo only)

Generator scripts: `create_tpt_packages.py`, `create_tpt_packages_find_cover.py`, `create_tpt_packages_bingo.py`

---

## 14. Output Requirements
- Color and BW versions for all products
- Review PDFs: `/review_pdfs/`
- Samples: `/samples/{theme}/{product_type}/`
- Final products: `/production/final_products/{theme}/{product_type}/`
- TpT ZIPs: `/production/final_products/{theme}/{product_type}/tpt_zips/`
- Marketing: `/production/marketing/{theme}/{product_type}/`
- Upload-ready: `/production/upload_ready/{theme}/{product_type}/`

---

## 15. Accessibility
- High contrast (white text on colored stripes)
- Clear iconography
- Predictable layout
- Errorless learning logic applied consistently
- Level colors provide instant visual differentiation
- AAC communication support in every Quick Start
