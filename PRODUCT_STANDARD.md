# Product Standard - Official Format for All Future Products

## 📋 Standard Product Format

**Official Standard:** Brown Bear Matching FINAL with cover

**This format is the standard for ALL future products across ALL themes.**

---

## 📄 Page Structure (17 Pages Total)

### Page 1: Professional Cover
**Features:**
- Level-colored accent strip (Orange/Blue/Green/Purple)
- Product title: "{Theme} {Product Type}"
- Level subtitle: "Level X • {Level Name}"
- Small theme image (2.5" × 2.5") in bordered box
- Product features list:
  - "15 Activity Pages"
  - "Level X Matching to Boards"
  - "Colour + Black & White Versions Included"
  - "Print-Ready Cutout Pieces (optional laminate/Velcro for reuse)"
  - "Bonus: Storage Labels Included"
- Quick Start instructions:
  - "Print → Cut → (optional laminate/Velcro) → Match pieces to boards → Pack away with storage labels."
- Footer with pack code, copyright, PCS® license

### Pages 2-16: Activity Pages (15 pages)
**Features:**
- Matching boards, cutout pieces, storage labels
- Page numbers in small text (e.g., "Page 2/17")
- Level-specific differentiation
- Design Constitution compliant:
  - Rounded borders (0.12" radius)
  - Navy borders (#1E3A5F)
  - Proper margins (0.5" all sides)
  - Comic Sans MS fonts
  - Level-colored accents

### Page 17: How to Use Guide
**Features:**
- Teacher instructions
- Setup guidelines
- PODD/AAC support tips
- Implementation strategies

## 🎨 Design Standards

### Level Colors (Must Match Throughout)
- **Level 1:** Orange (#F4B400) - Errorless
- **Level 2:** Blue (#4285F4) - Easy
- **Level 3:** Green (#34A853) - Medium
- **Level 4:** Purple (#8C06F2) - Challenge

### Typography
- **Primary Font:** Comic Sans MS
- **Fallback 1:** Arial Rounded MT Bold
- **Fallback 2:** Helvetica
- **Title:** 32pt bold
- **Subtitle:** 16pt
- **Body:** 12-14pt
- **Footer:** 9pt
- **Page numbers:** 8pt gray

### Layout
- **Page Size:** US Letter (8.5" × 11")
- **Margins:** 0.5" all sides
- **Borders:** Rounded rectangles, 0.12" radius, 2-3px width
- **Border Color:** Navy (#1E3A5F)
- **Accent Color:** Level-specific

---

## 📦 File Naming Convention

**Pattern:**
```
{theme}_{product}_level{number}_{levelname}_{format}_FINAL.pdf
```

**Examples:**
- brown_bear_matching_level1_Errorless_color_FINAL.pdf
- brown_bear_matching_level1_Errorless_bw_FINAL.pdf
- space_bingo_level2_Easy_color_FINAL.pdf
- space_bingo_level2_Easy_bw_FINAL.pdf

**Rules:**
- Lowercase theme and product
- Uppercase level name (Errorless, Easy, Medium, Challenge)
- Lowercase format (color, bw)
- UPPERCASE FINAL

## 📚 TpT Package Contents

**Each level package (ZIP) must contain exactly 4 files:**

1. **Color FINAL PDF** (17 pages with cover)
   - Complete product with all pages
   - Full color version
   
2. **B&W FINAL PDF** (17 pages with cover)
   - Complete product with all pages
   - Printer-friendly black & white version

3. **Terms of Use PDF**
   - Legal licensing information
   - Usage rights
   - Copyright protection

4. **Quick Start PDF**
   - Simple 1-page guide
   - Print → Cut → Match → Store
   - Visual workflow

**Do NOT include:** How to Use (already in main PDF page 17), Levels of Differentiation, More Packs

## 🎨 Marketing Materials

**Each product also needs (NOT in TpT ZIP):**

### Preview PDF
- Watermarked version for TpT preview
- First 3-5 pages with "PREVIEW" watermark
- Shows product quality without giving away everything

### Thumbnails
- **280×280 pixels** - TpT listing thumbnail
- **500×500 pixels** - TpT preview thumbnail
- PNG format
- Shows cover or sample page
- Eye-catching, professional

### Description
- TpT listing description text
- Features and benefits
- Page count and contents
- Grade levels and subjects
- Keywords for SEO

## 🔄 Generation Workflow

**Standard workflow for any new product:**

1. **Generate Core Product** (15 pages)
   - Activity pages with matching boards
   - Cutout pages
   - Storage labels
   
2. **Generate Cover** (1 page)
   - Use level-specific color
   - Include Quick Start
   - Add theme image
   
3. **Create Complete Product** (17 pages)
   - Merge: Cover + Core Product + How to Use
   - Add page numbers
   - Create color and B&W versions
   
4. **Create TpT Package**
   - Package: Color FINAL + B&W FINAL + TOU + Quick Start
   - ZIP file ready for upload
   
5. **Create Marketing**
   - Generate preview PDF
   - Create thumbnails (280×280, 500×500)
   - Write description

6. **Quality Check**
   - Verify all checklist items
   - Test print one page
   - Review all 17 pages
   
7. **Release**
   - Upload to TpT
   - Archive working files
   - Delete obsolete versions
