## Generated Files

### Matching Activities (10 files)

**Level 1-4 Activity Pages** (8 files)
- Color and B&W versions for each of 4 difficulty levels
- Each level follows Design Constitution standards
- Level-coded accent stripes (Orange, Blue, Green, Purple)

**Cutouts Page** (1 file)
- 5 icons per strip
- Ready for lamination and cutting

**Storage Labels** (1 file)
- 3-column vocabulary table
- Icons included for visual reference

## How Files Were Generated

These files were generated using:
```bash
python -m generators.matching --theme brown_bear --output exports/
```

Then copied to production folder for version control and distribution.

## File Specifications

- **Page Size**: US Letter (8.5" × 11")
- **Margins**: 0.5" on all sides
- **Border**: Rounded rectangle (0.12" radius)
- **Compliance**: Design Constitution and Master Product Specification
