# AAC/PCS Symbols Folder

This folder contains AAC (Augmentative and Alternative Communication) and PCS-style symbols.

## Guidelines

- **File Format**: PNG recommended
- **Style**: Simple, clear symbols following AAC/PCS conventions
- **Resolution**: At least 500x500 pixels
- **Background**: Transparent or white background
- **Naming**: Use the word/concept name (e.g., `eat.png`, `drink.png`, `happy.png`)

## Usage

These images are loaded using:
```python
folder_type='aac'
```

## Used By

- Sentence Strips (AAC) Generator
- Communication boards (if implemented)

## Examples

Place your AAC symbols here:
- Core vocabulary: `I.png`, `want.png`, `need.png`, `help.png`
- Actions: `eat.png`, `drink.png`, `play.png`, `sleep.png`
- Feelings: `happy.png`, `sad.png`, `angry.png`, `scared.png`
- Objects: Common nouns with clear, simple symbol representations

## About AAC Symbols

AAC symbols help non-verbal or minimally verbal students communicate. They should be:
- Clear and simple
- Easily recognizable
- Consistent in style
- Culturally appropriate

Note: If using PCS or other proprietary symbol sets, ensure you have proper licensing.
