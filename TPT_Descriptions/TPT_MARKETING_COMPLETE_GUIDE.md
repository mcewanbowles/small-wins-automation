## 📤 HOW TO UPLOAD TO TPT

### Step-by-Step Upload Process:

**1. Create Product Listing**
- Log into TPT Seller Dashboard
- Click "Create New Product"
- Select "PDF" as file type

**2. Upload Files**
- Upload ZIP file from `TPT_Products/`
- TPT will extract and show all 4 files
- Verify all files uploaded correctly

**3. Title (SEO-Optimized)**
Example: "Brown Bear Matching Activity - Level 1 | Special Education | Autism | AAC | File Folder Game"

**Format:** `[Product Name] - Level [X] | Special Education | Autism | AAC | [Activity Type]`

**4. Description**
- Copy full description from template above
- Paste into description field
- Adjust formatting if needed
- Add emoji for visual appeal
- Keep under TPT's character limit

**5. Set Price**
- Individual: $4.50
- Bundle: See pricing recommendations above

**6. Categories & Grade Levels**
- **Primary Category:** Special Education
- **Additional:** Speech Therapy, Life Skills
- **Grade Levels:** Pre-K, K, 1st, 2nd, Special Ed, Homeschool

**7. Add Tags (Maximum 10)**
Choose from high-value keywords:
1. special education
2. autism
3. AAC
4. file folder games
5. task boxes
6. [product-specific, e.g., "errorless learning"]
7. visual discrimination
8. matching activities (or activity type)
9. speech therapy
10. special needs

**8. Upload Preview Images**
- Use thumbnails from `Thumbnails/` directory
- 4-6 images recommended
- Include:
  - Cover page
  - 2-3 activity examples
  - Cutouts (if applicable)
  - Quick Start Guide preview

**9. Set Additional Options**
- ✅ Allow commercial use: NO
- ✅ Digital download: YES
- ✅ File type: PDF
- ✅ Pages: [actual page count]

**10. Publish!**
- Review all information
- Click "Publish"
- Product goes live on TPT
