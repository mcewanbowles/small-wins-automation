## Educational Features to Evaluate

### Progression Logic

The 5 levels follow a **scaffolding progression**:

1. **Most Support** (Level 1) → Visual hints with color
2. **Generalization** (Level 2) → Real photos connect to world
3. **Intermediate** (Level 3) → Removes color and realism
4. **Literacy** (Level 4) → Text-based, minimal visual
5. **Independence** (Level 5) → No help, full mastery

**Question:**
- [ ] Does this progression make sense for special education students?
- [ ] Would teachers find this useful?

### SPED-Friendly Design

**Snake Pathway:**
- Visual metaphor for "journey" through the story
- More engaging than grid layout
- Natural flow shows sequence

**Question:**
- [ ] Is the snake pathway engaging and helpful?
- [ ] Or would you prefer a different layout?
