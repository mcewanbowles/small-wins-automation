## PAGE 2: WHAT IS BODY ODOR?

**TEXT:**
```
MY BODY IS CHANGING

When I get older, my body changes.
One change is that my sweat can smell different.

Body odor means:
• My body's sweat smells stronger
• This happens to EVERYONE as they grow
• It means my body is working normally

Body odor happens when:
• I play sports or exercise
• I get hot or nervous
• My body goes through puberty
• I don't wash regularly

This is NORMAL!
All kids going through puberty experience this.

## PAGE 3: WHERE DOES BODY ODOR COME FROM?

**TEXT:**
```
WHERE BODY ODOR HAPPENS

Body odor can come from:
• Underarms (armpits)
• Feet
• Groin area (private parts)

These areas have special glands that make sweat.
When sweat mixes with bacteria, it creates odor.

I might notice:
• Smell after playing
• Smell when I take off my shoes
• Smell when I'm nervous or hot
• Stronger smell as I get older

This happens to EVERYONE!
```

## PAGE 4: HOW TO PREVENT BODY ODOR

**TEXT:**
```
TAKING CARE OF MY BODY

I can prevent body odor by:

1. SHOWER OR BATHE DAILY
   • Every day or every other day
   • Use soap on my whole body
   • Pay attention to underarms, feet, and private parts
   • Rinse well

2. WEAR CLEAN CLOTHES
   • Fresh shirt every day
   • Fresh underwear every day
   • Fresh socks every day
   • Wash gym clothes after each use

3. USE DEODORANT
   • Apply to clean, dry underarms
   • Every morning
   • After shower or bath
   • Reapply if needed

These habits keep me fresh and clean!

## PAGE 5: HOW TO USE DEODORANT

**TEXT:**
```
USING DEODORANT

Deodorant helps prevent body odor.

TYPES OF DEODORANT:
• Stick (solid)
• Roll-on (liquid)
• Spray
• Gel

All types work! Use what feels best.

HOW TO APPLY:

1. Start with CLEAN, DRY underarms
2. Swipe or spray deodorant
3. Let it dry before getting dressed
4. Apply every morning
5. Reapply after sports if needed

HOW MUCH?
• Stick: 2-3 swipes each underarm
• Roll-on: 2-3 rolls each underarm
• Spray: 2-second spray each underarm
• Gel: Small amount, spread evenly

Too much can feel sticky.
Too little won't work well.
Practice to find the right amount!
```

## PAGE 6: CHOOSING DEODORANT

**TEXT:**
```
FINDING THE RIGHT DEODORANT

There are MANY types of deodorant!

DEODORANT vs ANTIPERSPIRANT:
• Deodorant = stops smell
• Antiperspirant = stops sweat AND smell
• Either one works!

SCENTS:
• Unscented (no smell)
• Light scent (a little smell)
• Strong scent (lots of smell)

Try different ones to find what you like!

SENSITIVE SKIN?
Look for:
• "Sensitive skin" on the label
• Unscented
• Hypoallergenic
• Aluminum-free

WHERE TO APPLY:
✓ Underarms ONLY
✗ NOT on other body parts

WHEN TO REPLACE:
• Every 2-3 months
• When it's empty
• When it doesn't work anymore

Ask a parent or trusted adult to help you shop!
```

## PAGE 8: SOCIAL SITUATIONS

**TEXT:**
```
BODY ODOR & SOCIAL SITUATIONS

WHAT IF SOMEONE COMMENTS?

If someone says:
"You smell" or "Did you use deodorant?"

I can:
• Stay calm (they're trying to help)
• Say "Thanks for telling me"
• Go apply deodorant in private
• Take a shower when I get home

This is NOT mean - they're helping me know!

WHAT IF I NOTICE SOMEONE ELSE'S ODOR?

If I smell someone else:
• Don't point it out in public (this can hurt feelings)
• Tell a parent or teacher privately
• Let adults handle it
• Be kind - everyone forgets sometimes

HOW TO TELL SOMEONE KINDLY:

If I'm close friends:
"Hey, you might want to use deodorant"
Say it PRIVATELY, not in front of others

If it's not my close friend:
• Tell a parent, teacher, or counselor
• Let them handle it kindly

Remember: Everyone has body odor sometimes!
Being kind is most important.
```

## PAGE 9: TROUBLESHOOTING

**TEXT:**
```
COMMON PROBLEMS & SOLUTIONS

PROBLEM: Deodorant doesn't work
SOLUTION:
• Make sure underarms are clean and dry first
• Try antiperspirant instead
• Try a different brand
• Ask doctor about stronger options

PROBLEM: Deodorant irritates my skin
SOLUTION:
• Try "sensitive skin" formula
• Try unscented
• Apply to completely dry skin
• Ask parent about alternatives

PROBLEM: I forget to use deodorant
SOLUTION:
• Keep deodorant next to toothbrush (morning routine!)
• Set phone reminder
• Put sticky note on bathroom mirror
• Keep extra in backpack

PROBLEM: I still smell after using deodorant
SOLUTION:
• Shower more often (daily if needed)
• Change clothes daily
• Wash gym clothes after EACH use
• See doctor if shower + deodorant don't help

PROBLEM: I don't like the smell
SOLUTION:
• Try unscented
• Try different brands
• Ask parent to smell test with you

It may take time to find what works best!
```

## PAGE 11: PARENT/EDUCATOR GUIDE

**TEXT:**
```
GUIDE FOR PARENTS & EDUCATORS

TEACHING TIPS:

1. START EARLY:
   - Introduce deodorant around age 8-10
   - Before body odor becomes a social issue
   - Make it a normal part of growing up

2. NORMALIZE THE CONVERSATION:
   - Don't act embarrassed
   - Frame as "everyone does this"
   - Use matter-of-fact tone

3. PRACTICE TOGETHER:
   - Demonstrate application
   - Have child practice while you watch
   - Provide feedback kindly

4. CREATE ROUTINES:
   - Link to existing habits (after brushing teeth)
   - Use visual schedules
   - Set phone alarms if needed

5. SHOPPING TOGETHER:
   - Let child help choose scent/type
   - Start with affordable options ($3-5)
   - Buy 2-3 types to try

6. ADDRESS SENSORY ISSUES:
   - Try unscented if scents bother them
   - Test different textures (stick vs roll-on)
   - Apply to completely dry skin (reduces stickiness)
   - Consider sensitive skin formulas

7. SOCIAL AWARENESS:
   - Role-play what to do if someone comments
   - Teach private vs public application
   - Practice kind ways to tell others

8. TROUBLESHOOTING:
   - If deodorant alone isn't enough:
     * Increase shower frequency
     * Check for medical issues (hyperhidrosis)
     * Consider prescription options (ask doctor)
   - If they resist using it:
     * Explain WHY (social acceptance, comfort)
     * Show peer examples (older siblings, friends)
     * Make it non-negotiable like brushing teeth

9. FOR STUDENTS WITH AUTISM:
   - Use this social story multiple times
   - Create visual reminder cards
   - Use concrete language ("every morning")
   - Avoid vague language ("when needed")

10. SCHOOL SUPPORT:
    - Keep deodorant in backpack or locker
    - Create private reminder system with teacher
    - Use discrete cue if they forget
    - Don't shame or call out publicly

SIGNS IT'S TIME TO START:
• Body odor present
• Age 8-10 or older
• Early puberty signs
• Requests to use deodorant
• Increased sweat

RECOMMENDED PRODUCTS FOR BEGINNERS:
• Dove Sensitive Skin (gentle, unscented)
• Tom's of Maine (natural, various scents)
• Native (aluminum-free, many scents)
• Secret/Old Spice (affordable, effective)

MEDICAL CONCERNS:
Contact doctor if:
• Body odor is very strong despite good hygiene
• Skin reactions to all deodorants
• Excessive sweating (hyperhidrosis)
• Body odor before age 8 (early puberty sign)

Remember: This is a life skill that promotes independence and social acceptance!
```
