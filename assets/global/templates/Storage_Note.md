# Storage Note Template

> **FILE:** `Storage_Note.pdf`
> **INCLUDE IN:** Every product ZIP (can be combined with Storage Labels page)
> **PURPOSE:** Quick storage guidance

---

## STORAGE

Storage labels are included. Store pieces in a labelled pouch/envelope (file folder) or inside a labelled case (task box/photo case). Keep the product + level label on the container for quick set-up.

---

## Design Notes

- Can be a small section on the Storage Labels page
- Or a separate single paragraph page
- Include with icons showing:
  - File folder option
  - Task box/photo case option
- Keep brief — one paragraph only
