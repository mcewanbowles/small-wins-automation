# Student Directions Template

> **FILE:** `Student_Directions.pdf`
> **INCLUDE IN:** Every product ZIP
> **PURPOSE:** Simple visual instructions for students

---

## STUDENT DIRECTIONS

1) Look.

2) Choose one.

3) Match it.

4) Keep going.

5) Ask for help if needed.

6) Finished.

---

## Design Notes

- Use large, clear font (Comic Sans MS, 24pt+)
- Add simple icons next to each step if space allows
- Keep on ONE page
- High contrast (dark text on light background)
- Consider adding checkbox/tick boxes for each step
