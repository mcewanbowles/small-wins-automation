## Viewing Documentation

### On GitHub
1. Navigate to: `https://github.com/mcewanbowles/small-wins-automation`
2. Click on any `.md` file
3. GitHub renders markdown automatically

### In Local Repository
```bash
# View any document
cat PRODUCT_STANDARDS.md
cat README_THUMBNAILS.md
cat LOGO_INTEGRATION_GUIDE.md

# Or use a markdown viewer
code TOU_AND_DOCS_LOCATION_GUIDE.md
```

### In Editor
- Open repository in VSCode, Atom, or similar
- Browse to file
- Use markdown preview (Ctrl+Shift+V in VSCode)
