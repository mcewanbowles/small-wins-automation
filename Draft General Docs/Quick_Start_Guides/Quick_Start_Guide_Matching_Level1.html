Teaching support (prompting ladder)

Use the least help possible, then fade prompts over time:

  1. Independent (wait time)
  2. Gesture (point to the spot)
  3. Verbal cue ("match", "same")
  4. Model (place one piece)
  5. Physical support (only if needed)
Communication + AAC

🌟 AAC users: This resource is perfect for communication modeling! Model a few consistent words while matching using AAC e.g. My turn, Your turn, Same, Different, Again etc.

Ways to use this set (low-prep options)
  • Independent work station: board + pieces in a pouch, "finished" tray
  • Work box / task box: one board per box
  • File folder activity: board page in the folder, pieces in a pocket
  • 1:1 teaching: hand one piece at a time to build attention and success
  • Small group rotation: each student has a board; swap when finished
