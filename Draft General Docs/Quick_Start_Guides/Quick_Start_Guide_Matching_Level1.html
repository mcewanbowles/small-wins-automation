Matching
What this resource is

This set is Level 1: Errorless matching to boards. Students match and place each cut-out piece onto the same picture on the board page. Great for building early matching skills while supporting AAC use!

Part of a Differentiated Series

This is part of a multi-level Matching to Boards series! There are multiple levels available, each building on the last for differentiation and growth tracking. Save with the complete bundle!

What's included
  • 15 activity boards
  • Cut-out matching pieces
  • Colour + Black & White versions
  • Bonus: Storage labels
Prep (choose what suits your classroom)

You'll need: printer + scissors (or paper trimmer)
Optional: cardstock, laminator, Velcro dots, zip bags or folders

Set-up steps
  1. Print (Colour or B/W).
  2. (Optional) Print on cardstock for durability.
  3. (Optional) Laminate boards + pieces for long-term reuse.
  4. Cut the pieces.
  5. (Optional) Add Velcro (pieces on the back + boards on the matching spots).
  6. Store each board with its pieces (folder, pouch, or bag).
Student routine (independent station friendly)
  1. Get a board and the matching pieces.
  2. Match each piece to the same picture on the board.
  3. Fill all spaces.
  4. Check: point to each match (same/same/same).
  5. Pack away pieces and return the folder.
Teaching support (prompting ladder)

Use the least help possible, then fade prompts over time:

  1. Independent (wait time)
  2. Gesture (point to the spot)
  3. Verbal cue ("match", "same")
  4. Model (place one piece)
  5. Physical support (only if needed)
Communication + AAC

🌟 AAC users: This resource is perfect for communication modeling! Model a few consistent words while matching. Consistent modeling builds vocabulary naturally:

  • Action words: LOOK, MATCH, PUT, MOVE
  • Concepts: SAME, DIFFERENT, HERE
  • Help/repair: HELP, WHOOPS, TRY AGAIN
  • Finish: FINISHED, AGAIN
  • Turn-taking (optional): MY TURN, YOUR TURN
  • Checking: CHECK

(Keep it to 4–6 words per session so it stays calm and achievable.)

Ways to use this set (low-prep options)
  • Independent work station: board + pieces in a pouch, "finished" tray
  • Work box / task box: one board per box
  • File folder activity: board page in the folder, pieces in a pocket
  • 1:1 teaching: hand one piece at a time to build attention and success
  • Small group rotation: each student has a board; swap when finished
Quick games (same materials, more engagement)
  • Mystery Pick: student pulls 1 piece from a bag and finds the match
  • My Turn / Your Turn: take turns placing pieces
  • Find It: show 2–3 pieces; student finds the one that matches the board spot
  • Beat Your Best: time how long it takes to finish (only if motivating)
  • Mix & Choose: offer 2 boards; student chooses which one to do next
  • Check Routine: after finishing, student points to each match while you say "same"
Troubleshooting (teacher time-savers)
  • Pieces getting lost? Print one extra cut-out page and keep it as a backup.
  • Too hard right now? Use fewer pieces, cover extra spaces, or pre-place 1–2 matches.
  • Rushing? Add a simple "check" step before packing away.
