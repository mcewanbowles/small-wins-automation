## How to Use

### View in Browser
Simply open `Terms_of_Use_Credits.html` in any web browser.

### Convert to PDF
1. Open the HTML file in a web browser (Chrome, Firefox, Safari, etc.)
2. Use Print function (Ctrl+P or Cmd+P)
3. Select "Save as PDF"
4. Ensure the following print settings:
   - Paper size: Letter (8.5" × 11")
   - Margins: Default or None
   - Background graphics: Enabled (to show colors)

### Customization
To customize for a specific product:
1. Replace `[YOUR PRODUCT TITLE]` with your actual product title
2. Update `YOUR_STORE_URL` with your TPT store link
3. Adjust the copyright year if needed (currently 2026)
