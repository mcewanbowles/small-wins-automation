## How to Use

1. Print desired version (color or B&W)
2. Laminate for durability
3. Choose version based on student needs:
   - Use version with text for literacy development
   - Use version without text for pure AAC practice
   - Use white background for less distraction
4. Place on desk, table, or in communication book
5. Model use by pointing to symbols while speaking
