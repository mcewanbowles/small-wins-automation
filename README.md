# small-wins-automation

Automated TpT resource generators for Small Wins Studio

## 🎨 Product Generators

### Social Stories (NEW! ✨)
Generate beautiful, accessible social stories PDFs for SPED students covering sensitive topics.

**Features:**
- Professional PDF generation with Small Wins branding
- Large image placeholders for Boardmaker icons
- 5 stories ready to generate (Good Touch/Bad Touch, Puberty, Hygiene topics)

**Quick Start:**
```bash
pip install -r requirements.txt
python generators/social_stories/generator.py
```

See [generators/social_stories/README.md](generators/social_stories/README.md) for details.
