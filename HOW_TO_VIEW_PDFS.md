## 📖 Steps to View

### Option 1: View on GitHub (Easiest)

1. Go to your repository: `https://github.com/mcewanbowles/small-wins-automation`
2. Switch to branch: `copilot/update-python-code-colors`
3. Click on the `OUTPUT` folder
4. Click on any PDF file
5. GitHub will display the PDF in your browser

### Option 2: Clone and View Locally

```bash
git clone https://github.com/mcewanbowles/small-wins-automation.git
cd small-wins-automation
git checkout copilot/update-python-code-colors
cd OUTPUT
# Open PDFs with your preferred PDF viewer
```

### Option 3: Download Individual Files

1. Navigate to the OUTPUT folder on GitHub
2. Click on a PDF file
3. Click the "Download" button
4. Open with your PDF viewer
